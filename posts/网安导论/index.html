好，这里给你一份真正能应付期末的压缩版：《网络安全导论·密码学期末必背》🔥 （按“能直接默写/做题”的标准整理）

📘 一、密码学总基础（必背）

1. 密码学定义

  • 研究信息加密与破解的学科

  • 两部分：

    • 密码编码学（加密）
    • 密码分析学（破解）

2. 核心原则（最重要‼）

👉 现代密码学：安全性依赖密钥，而不是算法

3. 密码系统基本模型

  • 明文：M
  • 密文：C
  • 密钥：K

公式：

  • 加密：C = Eₖ(M)
  • 解密：M = Dₖ(C)

📘 二、对称密码体系（重点🔥）

分类（三大类）

类型 核心
传统密码 换字 / 换位
流密码 XOR
分组密码 分块 + 多轮

📘 三、传统密码（第2.1）

1. 两大类型

（1）代换密码

👉 改字符

  • 凯撒密码（移位）
  • 替换密码
  • Vigenère密码
  • Hill密码

（2）置换密码

👉 改位置

  • 栅栏密码

2. 核心弱点（必考‼）

👉 保持字母频率 → 可被统计攻击

3. 安全性排序（理解）

  • 单表代换 ❌
  • 多表代换 ✔
  • 多字母代换 ✔✔

📘 四、流密码（第2.2）

1. 核心公式（必背🔥）

[ C = M ⊕ K ] [ M = C ⊕ K ]

2. 本质

👉 伪随机密钥流 + XOR

3. 关键特点

  • 一位一位加密
  • 加密 = 解密

4. 一次一密（必考）

  • 密钥长度 = 明文长度
  • 完全随机
  • 理论不可破解

5. RC4（了解）

  • 流密码经典算法
  • 基于S盒
  • 已不安全

6. 流密码漏洞（常考）

👉 重复密钥： [ C1 ⊕ C2 = M1 ⊕ M2 ]

📘 五、分组密码（第2.3🔥重中之重）

1. 基本思想

👉 明文分块（如128bit）逐块加密

2. Shannon三大原则（必背‼）

✔ 扩散（Diffusion）

  • 明文变化 → 密文大变化

✔ 混乱（Confusion）

  • 密钥关系复杂

✔ 乘积密码

  • 多轮“代换+置换”

3. 雪崩效应（必考）

👉 输入改变1bit → 输出大幅变化

4. 分组密码设计要求（记3个）

  • 分组长度 ≥ 128bit
  • 密钥长度 ≥ 128bit
  • 多轮迭代（8~20轮）

📘 六、DES算法（必考🔥）

1. 基本参数

  • 分组：64bit
  • 密钥：56bit
  • 轮数：16轮

2. 核心结构

👉 Feistel结构

公式： [ L_i = R_{i-1} ] [ R_i = L_{i-1} ⊕ F(R_{i-1}, K_i) ]

3. 加密流程（记步骤）

  1. 初始置换
  2. 16轮迭代
  3. 逆置换

4. F函数（重点）

  • 扩展（32→48）
  • XOR密钥
  • S盒（非线性）
  • P盒（置换）

5. DES缺点（必考）

  • 密钥太短（56bit）
  • 可被暴力破解

📘 七、现代分组密码

必须知道：

  • DES（已淘汰）
  • AES（主流）
  • SM4（国产）

📘 八、核心对比（必背表🔥）

类型 本质 安全性
传统密码 换字/换位 ❌
流密码 XOR ⚠️
分组密码 数学结构 ✅

📘 九、考试高频问答（直接背🔥）

1. 为什么单表代换不安全？

👉 保持字母频率 → 可统计分析

2. 流密码为什么快？

👉 按位处理，无需分组

3. 什么是雪崩效应？

👉 输入微小变化 → 输出巨大变化

4. 分组密码核心思想？

👉 多轮代换 + 置换

5. 为什么DES不安全？

👉 密钥太短（可暴力破解）

6. 流密码核心风险？

👉 密钥流重复

📘 十、终极一句话总结（必背‼）

👉 传统密码：靠规则（不安全） 👉 流密码：靠随机（XOR） 👉 分组密码：靠结构（最安全）
